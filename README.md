# graphene_tr_arpes

This repository contains the code and data needed to reproduce the analysis in:

**Line shapes in time- and angle-resolved photoemission spectroscopy explored by machine learning**,  
Tami C. Meyer, Gesa-R. Siemann, Paulina Majchrzak, Thomas Seyller, Jennifer Rigden, Yu Zhang, Emma Springate, Charlotte Sanders, and Philip Hofmann,  
[arXiv:2506.02137](https://arxiv.org/abs/2506.02137)

The `environment.yml` file can be used to create a conda environment.  
The main code is in the `TR-ARPES_lineshapes_kmeans.ipynb` file.
